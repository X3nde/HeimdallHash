# Aviso sobre el repositorio

Este repositorio es público exclusivamente por exigencia académica, con el objetivo de facilitar la revisión del Trabajo Fin de Grado por parte del tribunal evaluador y del personal docente autorizado.

HeimdallHash no se distribuye como software open source ni como producto de libre uso. El acceso público al código no implica autorización para utilizar, modificar, copiar, redistribuir, explotar comercialmente, integrar en otros productos, desplegar en producción ni crear obras derivadas.

Para conocer las condiciones aplicables, consultar el archivo `LICENSE.md`.

## Finalidad del repositorio

El repositorio contiene una versión preparada para revisión académica del proyecto HeimdallHash, incluyendo:

* Código fuente del servicio HeimdallHash.
* Código fuente del configurador gráfico HeimdallhashConfigurator.
* Herramientas y scripts de generación de release.
* Scripts de instalación, arranque, parada y desinstalación.
* Scripts de validación funcional.
* Paquetes de prueba sintéticos.
* Resultados de validación académica.
* Documentación técnica y académica asociada.

## Confidencialidad

El proyecto se ha desarrollado respetando restricciones de confidencialidad. Por ello, el repositorio no debe contener:

* Nombres reales de sistemas.
* Nombres reales de servidores.
* Rutas internas reales.
* Credenciales.
* Certificados.
* Claves privadas.
* Datos de cliente.
* Información clasificada.
* Capturas sensibles.
* Paquetes documentales reales.
* Logs productivos.
* Configuraciones reales del entorno operativo.

Las rutas, centros, paquetes, configuraciones, evidencias y resultados incluidos en el repositorio corresponden a datos sintéticos o de laboratorio utilizados exclusivamente para validación académica.

## Uso de los paquetes de prueba

Los paquetes incluidos en `TestScripts` se proporcionan únicamente para reproducir o revisar la batería de pruebas funcionales del Trabajo Fin de Grado.

Estos paquetes no representan datos reales, no contienen información sensible y no deben utilizarse como base para entornos productivos.

## Advertencia de operación

HeimdallHash es un prototipo funcional desarrollado con finalidad académica. Aunque incluye scripts de instalación, ejecución como servicio Windows, generación de release y pruebas funcionales, no debe desplegarse en entornos productivos sin una revisión técnica, operativa, legal y de seguridad previa.
